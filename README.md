# 康貝特 200P 期末衝刺場

HTML5 Canvas 橫向闖關小遊戲。支援 **鍵盤** 與 **手機觸控**；關卡/流程以多個 `<script>` 檔案依序載入，並拆分在 `game/split/` 方便維護（保留同一個 global scope）。

## 立即遊玩（本機）

### 推薦：用本機伺服器開

在專案根目錄執行：

```bash
python3 -m http.server 5173
```

然後打開 `http://localhost:5173/`。

> 直接用 `file://` 打開 `index.html` 也能玩，但某些瀏覽器可能會讓影片/音效或資源載入行為不一致；遇到問題請改用本機伺服器。

## 操作

### 鍵盤

- **移動**：`←/→` 或 `A/D`
- **跳躍**：`↑` 或 `W` 或 `Space`
- **衝刺**：`Shift` 或 `J`
- **暫停/繼續**：`P` 或 `Esc`
- **重新開始**：`R`
- **跳到下一關**：`N`
- **切換音效**：`M`
- **互動/開始/進入下一段**：`Enter`

### 觸控 / 滑鼠

- **點一下畫面**：依遊戲狀態可用於開始、重開、或略過段落（例如：過場影片）
- **Stage 2**：拖曳瞄準、放開發射（slingshot）
- **手機觸控按鈕**：左/右、跳、衝（符合條件時顯示）
- **右上角 HUD**：`跳關`、`秒殺Boss`

## 專案檔案

- `index.html`：入口頁（含 `<base href="./">` 修正 GitHub Pages / 子路徑部署的資源路徑）
- `style.css`：版面/外框/觸控 UI/直向握機旋轉等樣式
- `game/split/`：拆分後的遊戲程式（**載入順序很重要**）
- `game/README.md`：拆檔與維護說明（給開發者）

## 開發說明（給要改程式的人）

這個專案目前刻意不改成 ES modules：`index.html` 以多個 `<script>` 依序載入拆分檔案，讓既有全域狀態/函式能保持運作。拆分檔案的分工與載入順序請見 `game/README.md`。
